Langage Mot clé Utilité
HTML p Sert à créer des paragraphes
br La séparation des lignes est importante
a Sert à créer des liens
td Sert à définir une cellule de données à l'intérieur d'un tableau
tr Définit une ligne de tableau au sein d'un tableau
CSS padding Sert à créer un espace entre le contenu d'un élément
border sert à définir et styliser le contour d'un élément
font-family Sert à spécifier le nom de la police de caractères à utiliser pour un élément de texte
font-size Sert à définir la taille de la police de caractères d'un élément
background-color Sert à définir la couleur de fond d'un élément
Javascript var Indique à JavaScript que vous déclarez une variable
true Est un mot-clé qui représente la valeur du type booléen pour le "vrai" logique
prompt Sert à demander une information à l'utilisateur via une boîte de dialogue modale dans le navigateur, incluant un message, un champ de saisie et les boutons OK/Annuler
while Sert à répéter des instructions lorsque le nombre d'itérations n'est pas connu à l'avance
function Son utilité principale réside dans la factorisation de code, permettant ainsi d'éviter la répétition, d'améliorer la lisibilité, et de structurer le code de manière modulaire pour une meilleure maintenabilité et débogage
